Pic Project

This is the culmination of 6 hours put in by Mark Hoffmann and Stuart Casorotto on Sept. 7th 2016. It utilizes the Imgur api to find top images in a number of categories. Select a category and vote which image / gif / meme you like better between a head to head battle. You can then see how different images are ranking against each other via votes on our site in the results page by hovering over the different bubbles.

If continued effort were to be put in, features we would add include: Grabbing the categories dynamically, Accounts, Better results visualizations, and Image and User preferences (machine learning clustering)
